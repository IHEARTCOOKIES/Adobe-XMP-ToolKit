Detailed Description

API for access to the XMP Toolkit utility services.

TXMPUtils is the template class providing utility services for the XMP Toolkit. It must be instantiated with a string class such as std::string. See the instructions in XMP.hpp, and the Overview for a discussion of the overall architecture of the XMP API.
